# ÉquiTeam 🏃

Générateur d'équipes équilibrées pour enseignants EPS.

## Fonctionnalités
- Gestion de groupes d'élèves
- Import CSV d'élèves
- Notation des élèves par critères personnalisés
- Génération d'équipes équilibrées (algorithme serpentin pondéré)
- Historique des séances
- PWA installable (hors-ligne)

## Lancer l'application
Ouvrir `index.html` dans un navigateur (ou via Live Server dans VSCode).

## Architecture
- `/js/groups/`     → Gestion des groupes
- `/js/students/`   → Gestion des élèves
- `/js/activities/` → Activités & critères
- `/js/session/`    → Séances & algorithme
- `/js/history/`    → Historique
